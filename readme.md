# Tainan T-Bike API Filter


## 網站簡介

一個好的服務都會有個強大的篩選功能，方便客戶篩選搜尋出想要的資料，舉凡旅館、活動、電商、線上課程皆能觀看到其應用。

此篩選器主題是 Tainan T-Bike 的站點查找，運用政府資料資料開放平台，建立簡單的站點查找服務。

[網站Demo](https://yangyangxun.github.io/Tainan-T-Bike/)

## 運用技術

1. JavaScript 
2. AJAX
3. JSON 資料解析
4. Bootstrap
5. HTML/CSS

## 主要功能

* 完整站點資訊顯示
* 行政區搜尋
* 站名搜尋
